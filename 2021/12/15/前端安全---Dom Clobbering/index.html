toString方法的特殊返回值

​ 前面说到，id属性值可以影响JS的相应变量值，当在JS中，将一个标签对象当成字符串处理的时候，会调用这个对象的toString方法。在HTML标签元素中，存在两个特殊的标签，即<a>和<area>标签。当在JS中调用这两个标签元素对应对象的toString方法的时候，会返回其href属性值。例如：4.6.3 API for a and area elements
